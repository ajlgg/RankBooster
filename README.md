# RankBooster
This project is going to be a long-term project web application that will allow Overwatch players or outside users to get AI-analyzed feedback on their gameplay from the games that they upload as a video file. 

**Watch as your rank skyrockets from the depths of bronze all the way to top500 just by utilizing this application to fully understand your gameplay, and fix the mistakes you are making**
